pub struct Scene<T: Entity> { /* fields omitted */ }

Scene is basically a manager for all entities and where to find them. It uses object pooling by instantiating a fixed number of entities at startup, in order to maintain a decent render speeds when creating and destoying entities during the game. Scene also provides tools for searching and borrowing spawned entities.

pub fn new(size: usize, factories: Vec<Box<dyn Factory<T>>>) -> Self[src]

Create a new Scene entity manager instance.

By setting the 'size' parameter, you preset the maximum amount of Object the new pool can hold and therefore spawn.

Factories are custom object factories that implement the Factory trait which can be called upon, by the Scene, when spawning a new object. Factories give the user the freedom to create different types of objects by customizing the output of these factories.
